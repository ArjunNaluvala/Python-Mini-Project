# Python Mini Projects

This repository contains small Python console-based projects created to practice core Python programming concepts.

These projects focus on real-world scenarios such as managing student records, banking operations, quizzes, libraries, and expense tracking.

## Projects Included

### 1. Student Management System
A console-based application to manage student records.

Features:
- Add student details
- View all students
- Search student by roll number
- Update marks
- Delete student record

Concepts Used:
- Lists
- Dictionaries
- Loops
- Functions
- Object Oriented Programming
- File handling

Data stored:
Roll Number, Name, Marks


### 2. ATM / Bank Account Simulation

A simple banking system simulation that performs basic ATM operations.

Features:
- Deposit money
- Withdraw money
- Check account balance

Concepts Used:
- Classes
- Conditional statements
- Loops
- Functions
- Basic banking logic


### 3. Quiz Application (MCQ System)

A simple multiple-choice quiz system.

Features:
- Ask questions to the user
- Evaluate answers
- Calculate score

Concepts Used:
- Lists
- Dictionaries
- Conditional statements
- Functions


### 4. Library Management System

A small system to manage books in a library.

Features:
- Add new books
- Issue books
- Return books
- View available books

Concepts Used:
- Lists
- Loops
- Conditional statements
- Object Oriented Programming

---

### 5. Expense Tracker

A simple application to track daily expenses.

Features:
- Add expenses
- View expense records
- Calculate total expenses

Concepts Used:
- Lists
- Loops
- Functions
- Data tracking


## Technologies Used

Python

Concepts practiced:
- Variables
- Lists and Dictionaries
- Loops
- Conditional Statements
- Functions
- Object Oriented Programming
- Basic File Handling



## Learning Outcome

These projects helped practice Python fundamentals and understand how programming can be used to build small real-world systems.
